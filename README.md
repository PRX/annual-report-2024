# Annual Report 2024
Jekyll site for the single page PRX 2024 Annual Report

## Local Development

Clone the Github repo
* `$ gem install bundler jekyll`
* `$ gem install vips`
* `$ bundle install`
* `$ bundle exec jekyll serve`
* Go to [http://127.0.0.1:4000/](http://127.0.0.1:4000/) in your browser
